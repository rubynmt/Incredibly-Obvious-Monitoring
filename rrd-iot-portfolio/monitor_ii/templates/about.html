Components

  • Monitor Component

    Location:
    monitor
    Executable code:
    monitor.py
    Description:
    Monitor is the code that runs and monitors the various sensors connected to the Raspberry Pi. It updates every 5 seconds, storing data in an SQLite database.
    Running:
    python monitor.py

  • Data Provider

    Location:
    monitor
    Executable code:
    app.py
    Description:
    App is a simplified miniature Flask based API application that allows the monitored data to be queried by remote dashboard applications.
    Running (Debug):
    export FLASK_APP=app
export FLASK_ENV=development
flask run
    Running (Test Production):
    export FLASK_APP=app
export FLASK_ENV=production
flask run

  • Dashboard Component

    Location:
    dashboard
    Executable code:
    app.py
    Description:
    Dashboard presents the user with a menu containing options for each sensor and a front page with a summary of current data. Each page has dynamic display of the data in a chart.
    Running (Debug):
    export FLASK_APP=app
export FLASK_ENV=development
flask run
    Running (Test Production):
    export FLASK_APP=app
export FLASK_ENV=production
flask run
